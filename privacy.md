本機器人提供之功能包含：協助伺服器管理者進行濫用偵測、檢舉處理與違規處分之社群安全與審核服務；於用戶主動提及（mention）時提供之 AI 對話服務；以及供授權管理者主動匯出特定頻道訊息以供留存之功能，該匯出結果僅提供予執行該操作之授權管理者。

### 二、個人資料的蒐集、處理及利用方式

當您使用本機器人或加入本機器人存在之伺服器時，本機器人將視功能性質，請您或 Discord API 提供必要之個人資料，並在該特定目的範圍內處理及利用您的個人資料；非經您同意，本機器人不會將個人資料用於與其運作無關之其他用途。

本機器人秉持**資料最小化**之原則，僅於必要時讀取訊息內容，並依下列方式處理：

* **一般訊息（被動偵測）**：為偵測濫發、洗版等濫用行為，本機器人會即時讀取伺服器中之訊息以判斷是否出現異常模式。此項讀取不儲存訊息內容，僅於記憶體中暫存不含完整原文之比對特徵，且該特徵於約 60 秒後自動清除。未偵測到濫用行為時，相關資料即被拋棄，不作任何保存。

* **檢舉與處分（事件觸發）**：當用戶檢舉訊息或管理者執行審核操作時，本機器人會讀取相關訊息及其前後脈絡以協助判斷，並將相關訊息產生為證據圖片與脈絡檔案，張貼於該伺服器之管理用頻道或討論串。本機器人之資料庫不儲存任何訊息原文，僅儲存事件相關之識別資訊（例如用戶 ID、訊息 ID、頻道 ID、處分類型、理由及時間）。

* **AI 對話（第三方處理）**：當您主動提及本機器人進行對話時，您的訊息內容（包含隨訊息提供之文字與圖片）會傳送至 Google 之 Gemini API 進行處理以產生回應，此部分資料之處理適用 Google 之隱私政策。本機器人僅記錄此類請求之用量統計（例如用戶 ID、token 數量與時間）以供運作管理，不保存對話之訊息內容。若您不希望訊息被傳送至第三方處理，請勿主動提及本機器人。

本機器人在您使用其功能時，可能保留您之帳號名稱、帳號暱稱、帳號頭像及帳號識別碼等必要資料。

為提供精確之服務，本機器人可能將蒐集之資料進行統計與分析，分析結果之統計數據除供內部研究外，將視需要公佈，但不涉及特定個人之資料。

上述資料之保留方式如下：

| 資料類型 | 保留方式 |
|----------|----------|
| 一般訊息（被動偵測特徵） | 約 60 秒後自動自記憶體清除，不持久化 |
| AI 對話內容 | 不保存（僅即時傳送至 Google 處理） |
| AI 用量統計 | 保存於資料庫 |
| 檢舉紀錄 | 長期保存 |
| 處分紀錄（含已撤銷之紀錄） | 永久保存，以供管理一致性與申訴查證 |
| 證據圖片與脈絡檔案 | 保存於 Discord 平台之管理頻道或討論串 |

當本機器人退出伺服器時，本機器人資料庫中與該伺服器相關之資料將予以清除。惟已張貼於 Discord 頻道之證據產物屬該伺服器之內容，其存續由該伺服器與 Discord 管理。
